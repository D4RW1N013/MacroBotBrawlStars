# MacroBotBrawlStars
🧠🎮 BlueStacks AFK Macro Setup Guide


🇬🇧 English Guide

⬇️ Scroll down for the German guide.

✅ Requirements
BlueStacks installed
Brawl Stars installed and running
Python installed

👉 Install Python via Command Prompt:

winget install Python.Python.3
⚙️ Setup
1. Download Files
BlueStacks_Macro
Start_BlueStacksBatch
afk_bs.json
2. Prepare Folder

Put all files into the same folder.

3. Run as Administrator ⚠️

Right-click on Start_BlueStacksBatch
➡️ Click "Run as Administrator"

❗ Required or it won’t work.

4. Load Macro
Program opens
Click "Load" (bottom right)
Select afk_bs.json

💡 Pre-built AFK loop included.
